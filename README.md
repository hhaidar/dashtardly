# dashtardly

> A react component library to create beautiful TV dashboards

[![NPM](https://img.shields.io/npm/v/dashtardly.svg)](https://www.npmjs.com/package/dashtardly) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save dashtardly
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'dashtardly'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
